# Setup

Welcome! Before we get started, make sure you have set the two required secrets:

- **SBC_AGENT_ID** — your agent ID from [agents.stablecoin.xyz](https://agents.stablecoin.xyz) → API Keys panel
- **SBC_AGENT_API_KEY** — your API key from the same panel

Once the secrets are set, fund your agent wallet by sending SBC to the address I'll share with you after your first payment or balance check.

You're ready to go. Try asking:
- "What's my SBC balance?"
- "What's my wallet address?"
- "Pay for https://api.example.com/data"
